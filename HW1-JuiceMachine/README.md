# Fruit Juice Ordering System

This is a beginner Python project simulating a CLI-based juice ordering system.

## 🌟 Fun Fact
The greeting message (`skrrt!`) contains a pop-culture reference inspired by **Squid Game** and **Thanos** memes.

## Features
- **User Interaction:** Greets the user and takes orders via console input.
- **Menu Logic:** Handles various fruit options (Apple, Orange, etc.) and sizes.
- **Pricing Algorithm:** Calculates total price based on fruit type, cup size, and extras (ice/sugar).
- **Special Conditions:**
  - Includes surcharges for "rare combinations".
  - Contains hidden logic/discounts for specific users (e.g., "ALEX").
  - Handles invalid inputs gracefully using `while` loops.

## Technologies Used
- Python
- Control Flow (`if/elif/else`)
- Loops (`while True`)
- Input validation
