# Advanced Data Structures: Trees & Recursion

This assignment expands into more complex algorithms, focusing on custom Linked List sorting, Binary Tree traversals, and recursive problem-solving.

## Features

- **Linked List Mastery:** Implementation of QuickSort on linked lists, merging multiple lists, and safely removing duplicates.
- **Tree Operations:** Custom Tree class methods that can:
  * Check equality between two different trees.
  * Invert a tree's structure.
  * Find the exact path between two specific nodes.
  * Execute an Euler Tour Traversal.
- **Functional Recursion:** Clean, recursive functions designed to:
  * Generate all possible subsets of a given set.
  * Calculate all possible permutations of a specific string.
  * Generate a geometric sequence mathematically.

## Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- Recursion & Algorithmic Sorting
