# City Search Algorithms & Database

This project implements fundamental search algorithms and manages a city database using Python dictionaries.

## Features
- **Search Algorithms:** Implements and compares four types of search:
  1.  Linear Search (Iterative)
  2.  Linear Search (Recursive)
  3.  Binary Search (Iterative)
  4.  Binary Search (Recursive)
- **Data Management:**
  - Loads city data (ID, Name, Population, Tags) from `cities.txt`.
  - Supports CRUD operations: Adding/Removing cities and tags.
  - Uses specific data structures (Sets for tags, Dictionaries for cities).

## Technologies Used
- Recursion & Iteration
- Binary Search Algorithm
- Python Dictionaries and Sets
- File Parsing
