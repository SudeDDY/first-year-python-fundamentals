# Custom ADTs: Stacks, Queues & Sets

This project focuses on building fundamental Abstract Data Types (ADTs) entirely from scratch, without relying on Python's built-in implementations or external libraries.

## Features

- **Palindrome Checker:** Utilizes a custom Stack ADT to verify if a given string reads the same forwards and backwards.
- **Queue Filtering:** Takes a stream of integers and cleanly separates them into positive and negative Queues, discarding zeroes.
- **Linked List Sets:** A custom representation of mathematical Sets using Linked Lists. Includes functionality to:
  * Insert elements (preventing duplicates).
  * Verify if two sets are disjoint.
  * Perform mathematical Union, Intersection, and Difference operations.

## Technologies Used

- Python 3
- Abstract Data Types (ADT)
- Linked Lists
